# SkillBridge

**SkillBridge** is a community-driven platform designed to help students build real tech skills and get hired through **Incremental Learning**.

## The Philosophy: Incremental Learning

Our core methodology is built on a simple yet powerful idea: **we start from the basics as we grow further, and we never forget the past.** We learn by combining the past with the current.

Instead of isolating topics, we continuously stack and mix concepts:
- **Phase 1:** Learn `Concept 1`
- **Phase 2:** Learn `Concept 1 + Concept 2`
- **Phase 3:** Learn `Concept 1 + Concept 2 + Concept 3`

This compound learning approach ensures that foundational skills remain sharp, and you understand how different technologies interact in real-world scenarios. It's not about watching isolated tutorials; it's about building, breaking, fixing, and mixing concepts together.

## Features & Impact
- **No Spoon-Feeding**: Real learning happens through struggle, honest reflection, and curiosity.
- **Community-Driven**: Join 2,400+ students from tier-3 colleges all figuring it out together.
- **Daily Coding Practice**: Continuous practice, live daily sessions, and peer feedback.
- **Free for Everyone**: 100% free to join, no hidden fees.

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/Sjking2025/skillbridge.git
   ```
2. Navigate to the project directory
   ```bash
   cd skillbridge
   ```
3. Install dependencies
   ```bash
   npm install
   ```

### Running Locally
To start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
